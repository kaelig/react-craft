# Team Context

This skill defines the react-craft multi-agent team: who does what, how they collaborate, and the shared values that guide every decision.

- **[charter.md](./charter.md)** — Shared values, collaboration rules, and project context that apply to all agents.
- **[roster.md](./roster.md)** — The seven specialized agents, their responsibilities, and their pipeline order.

Every agent in the pipeline reads these documents before starting work. They establish the common ground that keeps handoffs clean and output consistent.
